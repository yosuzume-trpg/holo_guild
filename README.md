# ホロギルド

半放置型ギルド運営RPG

## コンセプト

プレイヤーはギルドマスターとなり、ギルドを発展させながらダンジョン攻略を行う。
少数の主力キャラクターでダンジョンを攻略し、多数のキャラクターを生産活動へ配置することで資金を獲得する。
獲得した資金はキャラクター募集、キャラクター強化、装備強化、ギルド強化へ使用する。

# オフライン

オフラインでの放置は8時間を上限とします。オフライン中に経過した時間に応じて配置したキャラクターの報酬などを得ます。ただし、サイクルは経過しません。

# サイクル

ゲーム内時間としてサイクルがあります。20分ごとに1サイクルです。
サイクルの開始時、5%の確率で生産施設の素材のうち1つが豊作になり、+1%の生産量を追加します。例えば小麦生産量+1％などです。豊作ボーナスはそのサイクルの間のみ有効で、次のサイクル開始時にリセットされます（累積しません）。
また、キャラクターとの親愛度を上昇させるためのコンテンツは1サイクルでリセットされます。
サイクルが進むのはオンライン中のみです。

# ギルド開設(ゲーム開始)

初期所持金は1000G、最初に好きなキャラクターを1体選択します。選択したキャラクターには鉄の剣と冒険者の服が与えられます。これは最初に選択したキャラクターのみで、以降募集などで獲得したキャラクターには武器や防具は与えられません。
開始時に解放されている募集地域は、最初に選んだキャラクターの所属地域1つのみです。

ギルド全体を管理する施設。
ギルド全体のレベルとして、ギルドランク(GuildRank、 略称GR)を持っています。

キャラクター管理画面。

## 募集(Offer)

資金を消費してキャラクターを募集します。
利用できる地域はGR上昇のたびに1つ追加で解放でき、好きな地域を選択します。
募集するには地域を選択し、300Gのコストを支払うことで新しいキャラクターを獲得します。所持しているキャラクター数(証書は含めません)に応じてコストは高騰します。20種ごとに500×floor(所持数/20)上昇します。たとえば1〜19人なら300G、20〜39人なら800G…と500Gずつ上昇します。
選択した地域に応じたキャラクターを獲得可能です。地域は全5種類あり、一つの地域につき20名のテーブルがあります。そのテーブルからランダムにキャラクターが選ばれます。選ばれるキャラクターは重複します。重複した場合は「〇〇の証書」を獲得します。〇〇にはキャラクター名が入ります。ユーザーは〇〇の証書を利用してキャラクターの★ランクを上昇させます。
登場するキャラクターの★ランクは1固定です。事実上レアリティはないので、全員が同じ確率で排出されます。
1回募集するごとにその地域の募集ポイントがたまり、60募集ポイントたまるとその地域の好きなキャラクターを選択できます。すでに所持しているキャラクターを選択した場合、そのキャラクターの証書を5枚獲得します。

## 商人ギルド

商品販売施設。
集めた素材や制作したアイテムを売却します。手動で販売することもできますが、キャラクターを配置することで自動で売却することも可能です。配置されたキャラクターは何を売るか、最低何個の商品を残しておくか(最低保有数)を設定できます。
自動の場合、1分あたり1個を売却します。手動の場合はクリックするだけで即時売却します。
商人ギルドの拡張をすることで配置できるキャラクター数を増加させることができます。初期値は3名で、1レベル当たり1名増加します。コストは500×(ギルド拡張レベル^2)とします。
また、商人ギルドの研究をすることで販売スピードを上昇させることができます。初期値は1倍速で、1レベル当たり+1%します。例えば商人ギルドの研究レベルが2ならば1.01倍速、レベル3で1.02倍速となります。同様にコストは500×(ギルド研究レベル^2)とします。
拡張と研究の強化上限はGR×10までとします。GRが2の場合、20まで強化できることになります。
また、キャラクター単位でキャラクター商人レベルでのボーナスもあります。

## 工芸ギルド

素材からアイテムを製作する施設。
生産施設で集めた素材を商品化します。商品化する手間を加えることで、素材のまま販売するより高く売却したり、戦闘時に回復アイテムとして使用することもできます。キャラクターを配置し、制作するアイテムを設定することで自動でアイテムを売却できます。
自動の場合、1分あたり1個を制作します。手動の場合はクリックするだけで即時制作します。
工芸ギルドにも拡張と研究があり、拡張することで配置できるキャラクター数を増加させることができます。初期値は3名で、1レベル当たり1名増加します。コストは500×(ギルド拡張レベル^2)とします。
研究することで製作スピードを上昇させます。1レベルあたり+1%します。5ならば1.05倍速でアイテムを製作します。コストは500×(ギルド研究レベル^2)とします。
また、キャラクター単位でキャラクター工芸レベルでのボーナスもあります。

## 生産施設

各施設にはキャラクターを配置できます。初期値で3名配置可能です。施設を拡張することで、1名を追加できます。拡張コストは500×(拡張レベル^2)とします。上限はGR×10までとします。各施設にキャラクターを配置したあと、どの素材を生産するかの人数配分を設定します。配分設定は人数だけなので、どのキャラクターがどの素材を生産しているかまでは見ません。
研究することで1分あたりの生産量を+1%します。研究コストは500×(研究レベル^2)とします。上限はGR×10までとします。この研究ボーナスはキャラクター生産レベルのボーナスと加算で重複します（例: 研究レベル3=+3% ＋ キャラ生産レベル相当ぶん）。
施設ランクのほかに、キャラクターごとに各施設レベルを持ちます(農業レベル･鉱業レベル･･･など)。
また、キャラクター生産レベルのボーナスもあります。
100×施設レベルの経験値が溜まると、施設レベルが上昇します。1レベルあたり+1%生産量が上昇します。施設レベルの上限は施設ランク×10までです。
手動で生産する場合は1クリックあたり1G分生産します。1Gの素材は1クリックで生産できますが、10Gの素材は10クリックが必要です。なお、手動生産では経験値は得られません（キャラクター生産レベルの経験値は配置による自動生産でのみ加算されます）。同様に商人ギルド・工芸ギルドの手動操作でも経験値は加算されません。

## 鍛冶・仕立屋ギルド

同じ武器・防具・アクセサリー同士を合成し強化します。
鍛冶は武器・道具、仕立屋は防具・アクセサリーを強化します。
合成するためには同じ★ランク同士である必要があります。また、武器の合成には魔物の牙が★ランク×20個、防具の合成には魔物の皮★ランク×20個必要です。★3から★4にするには60個必要になります。
武器防具は★ランクがあがるごとに+25%します。例えば★1で+50%ならば★2で+75%、★3で+100%、★4で125%となります。防具のように複数ステータスが上昇する場合、複数のステータスで+25%されます。たとえば防御50%・魔法防御力25%→防御力75%・魔法防御力50％です。
アクセサリーの場合は★ランクがあがるごとに+5%します。
道具(生産量上昇)の場合は2倍の効果にします。★1で1%、★2で2%、★3で4%、★4で8%となります。

# キャラクター

## 基本方針

全キャラクターが全役割を担当可能にします。どのキャラクターにどの役割をさせるかは、ユーザーの自由です。すべてのキャラクターが戦闘・農業・鉱業・漁業・錬金・工芸・販売が可能です。
最終的なキャラクター数は100を目標にさせ、100人いたら終盤のプレイ時間となるようにします。

---

## 戦闘レベル・経験値

戦闘経験値で上昇します。必要経験値は100×戦闘レベルです。戦闘レベルの上限はGR×10までです(放置で極端に強くなりゲームクリアが簡単になるのを防ぐため)。
1戦闘レベルにつき、
HP 200±15
攻撃 50±5
防御 50±5
魔力 50±5
魔法防御 50±5
素早さ 10±1
だけ増加します。

### ステータス

初期ステータスはランダムですが、キャラクターごとに「傾向」があり、得意なステータスの振れ幅が大きくなります。

| 傾向     | HP  | 攻撃 | 防御 | 魔力 | 魔法防御 | 素早さ |
| -------- | --- | ---- | ---- | ---- | -------- | ------ |
| 標準型   | ±15 | ±5   | ±5   | ±5   | ±5       | ±1     |
| 攻撃型   | ±15 | ±15  | ±2   | ±2   | ±2       | ±1     |
| 魔法型   | ±15 | ±2   | ±2   | ±15  | ±2       | ±1     |
| 防御型   | ±20 | ±2   | ±15  | ±2   | ±15      | ±1     |
| 素早さ型 | ±15 | ±5   | ±5   | ±5   | ±5       | ±3     |

平均値はすべての傾向で同じです。傾向は運良く高い値が出ればそのキャラクターの個性になりますが、プレイヤーは好みのキャラクターを自由に編成できます。傾向はスカウト（募集）時にランダム決定します。

## キャラクター生産レベル

各生産施設ごとに存在します。
各施設に配置されたキャラクターは、1分毎に生産した素材の価格と同量の経験値を得ます。例えば1分で2.5G分の生産を行った場合、2.5経験値を得ます。1キャラクター生産レベルごとに+0.5%されます。
必要経験値は100×キャラクター生産レベルです。

## キャラクター商人レベル

商人ギルドに配置されたキャラクターは、1個販売するたびに1経験値を得ます。例えば1分で2.5個販売した場合、2.5経験値を得ます。1キャラクター商人レベルごとに販売速度が+1%されます。
必要経験値は100×キャラクター商人レベルです。

## キャラクター工芸レベル

工芸レベルに配置されたキャラクターは、1個製作するたびに1経験値を得ます。例えば1分で2.5個製作した場合、2.5経験値を得ます。1キャラクター工芸レベルごとに制作速度が+1%されます。
必要経験値は100×キャラクター工芸レベルです。

## キャラクター凸

同一キャラクターの〇〇の証書を合成します。要求数は凸ごとに倍になります。上限はありません。コストは500×★数とします。
1凸ごとに生産量が+2%され、戦闘ステータスが+5%されます。

### 例

```text
★1 +証書 → ★2
★2 + 証書×2 → ★3
★3 + 証書×4 → ★4
```

## 強化

同レアリティ装備同士を合成し、強化します。
キャラクターと違い、装備は複数人が装備するので証書のようなアイテム化はせず、同一装備を複数持つことができます。そのため、同じ★ランク同士でかけ合わせることで装備を強化します。強化費用は1000G×★です。★ランクに上限はありません。

### 例

```text
★1武器 + ★1武器 → ★2武器
```

## 装備

武器・道具・防具・アクセサリーを装備可能です。各項目で1つ、合計4つまで装備できます。

## 装備制限

キャラクター★以下の装備のみ装備可能。★3キャラクターならば、★3以下の装備しか使用できません。

## 親愛度・親愛ポイント

初期値は親愛度は1です。
1サイクルにつき全体で1回だけキャラクター1名と「交遊」をすることができます。交遊はキャラクターを選んでクリックするだけで完了し、そのキャラクターのカットインが表示されます。交遊をすることで、1回につき100±5親愛ポイントを獲得します。親愛度×100の親愛ポイントを獲得することで、親愛度が1上昇します。親愛度はステータスなどには影響しません。
10親愛度ごとに衣装を解禁し、5親愛度でホーム画面に表示できます。

# ダンジョン

## 基本構成

1人〜5人のパーティーを選択し、1ダンジョンにつき6ステージをクリアします。
すべてが戦闘ではなく、宝箱部屋、回復部屋などのイベントがあります。
|ステージ|種別|補足|
|1|戦闘（固定）|軽い導入戦|
|2|戦闘60% / 宝箱40%||
|3|戦闘（固定）分岐発生の候補||
|4|回復50% / 宝箱30% / 戦闘20%|ボス前の補給枠|
|5|戦闘（固定）|ボス前、分岐「通常/強敵」候補|
|6|ボス（固定）||

1ステージあたりの敵は1～3体です。ダンジョンレベルで分岐させます。
1～15　1～2体
16～40 2～3体
41～ 2～3体、強敵ステージ+1
敵タイプの配分は標準が40％・攻撃型25%・魔法型25%・防御型10%にします。
敵の属性はダンジョンレベル mod 4で火/風/地/水をループします。ボス・強敵は必ずそのダンジョン属性を持ちます。通常敵は約25%の確率で無属性のままとなり、残りはダンジョン属性を持ちます。

宝箱の中身は50G×ダンジョンレベル、60%でダンジョン素材のいずれかを5～10個、25%でポーション、12%でランダムな装備とします。

強敵を撃破すると150G×ダンジョンレベル、ランダムな装備、ダンジョン素材のいずれかを10~15個得ます。

回復ステージではHPを最大HPの50%回復させます。

## 全滅・撤退

HPはステージ間で引き継ぎます。ただしダンジョンから戻った際（クリア・撤退・全滅のいずれの場合も）、パーティーキャラクターのHPは全回復します。

**全滅**: パーティー全員がHPゼロになった場合、挑戦失敗・報酬なし。次回ダンジョン挑戦時はパーティー全員が全回復した状態でスタートします。

**撤退**: ステージ開始前にいつでも撤退できます。撤退した場合、現時点で獲得した報酬と経験値の50%を獲得します。

## 戦闘システム

セミオートコマンドバトルとなります。素早さ順に行動し、自分のパーティーの手番がきたら攻撃・魔法・アイテムから選択します。

攻撃を選択した場合、さらに対象を選択します。その後、自分の攻撃力と対象の防御力を比較し、ダメージを与えます。ダメージ計算式は、`自分の攻撃力^2 / (自分の攻撃力 + 相手の防御力)`とします。小数切り捨て・最低ダメージ1です。

魔法を選択した場合、さらに対象を選択します。その後、自分の魔力と対象の魔法防御力を比較し、ダメージを与えます。ダメージ計算式は、`自分の魔力^2 / (自分の魔力 + 相手の魔法防御力)`とします。小数切り捨て・最低ダメージ1です。

また、属性を考慮します。属性は火・水・風・地があり、火＜水＜地＜風＜火のような関係を持ちます。例えば、火は水に弱く風に強い、水は地に弱く火に強い、という関係です。強い場合を有利属性、弱い場合を不利属性とします。

キャラクター自身は属性を持ちません。攻撃時は装備している武器の属性、被攻撃時は装備している防具の属性を参照します。武器または防具を装備していない場合は無属性として扱い、有利・不利は発生しません。もし攻撃側の属性が有利属性ならば、ダメージ計算後に1.5倍されます。もし不利属性ならば0.75倍されます。

アイテムを選択した場合、どのアイテムを誰に使用するかを選択します。使用できるアイテムはダンジョン入場時に持ち込んだ工芸品(薬品)のみです。持ち込みは各種5個まで、合計制限なしとします。

### アイテム効果

| 名前           | 効果                  | 対象    |
| -------------- | --------------------- | ------- |
| 回復ポーション | HP +25% 回復          | 味方1人 |
| 強壮剤         | 攻撃力 +10% (3ターン) | 味方1人 |
| 泡薬           | 防御力 +10% (3ターン) | 味方1人 |
| 魔力薬         | 魔力 +10% (3ターン)   | 味方1人 |
| 万能薬         | HP +50% 回復          | 味方1人 |

バフは同じ種類を重ねがけした場合、ターン数をリセット(延長)します。異なる種類のバフは同時に適用されます。

### 敵AIの行動パターン

| 敵タイプ     | 使用する攻撃                 | ターゲット選択         |
| ------------ | ---------------------------- | ---------------------- |
| 通常・標準   | 物理攻撃                     | ランダム               |
| 通常・攻撃型 | 物理攻撃                     | ランダム               |
| 通常・魔法型 | 魔法攻撃                     | ランダム               |
| 通常・防御型 | 物理攻撃                     | 現在HPが最も低いキャラ |
| 精鋭（強敵） | 攻撃・魔力のうち高い方を使用 | 重み付きランダム       |
| ボス         | 攻撃・魔力のうち高い方を使用 | 重み付きランダム       |

**重み付きランダム**: 強敵・ボスが物理攻撃を行う場合は防御が低いキャラほど、魔法攻撃の場合は魔法防御が低いキャラほど選ばれやすくなります。各キャラのウェイトは `1 / 対応防御値` で計算し、確率を正規化します。完全にランダムではなく、やや弱点を狙ってくる程度の賢さです。

戦闘に勝利した場合、戦闘経験値を得ます。1体につき20×ダンジョンレベル、強敵は50×ダンジョンレベル、ボスは100ダンジョンレベルの戦闘経験値を手に入れます。手に入れた戦闘経験値はパーティ全員に同じだけ入ります。

敵の基礎ステータスは1ダンジョンレベル(D)あたり、
| 種別 | HP | 攻撃 | 防御 | 魔力 | 魔防 | 速 | 戦闘XP |
| 通常・標準 | 125D | 75D | 40D | 75D | 40D | 8D | 20D |
| 通常・攻撃型 | 125D | 90D | 50D | 30D | 25D | 10D | 20D |
| 通常・魔法型 | 125D | 30D | 25D | 90D | 50D | 10D | 20D |
| 通常・防御型 | 150D | 40D | 65D | 40D | 65D | 5D | 20D |
| 精鋭（強敵） | 270D | 95D | 40D | 95D | 40D | 9D | 50D |
| ボス | 600D | 110D | 40D | 110D | 40D | 10D | 100D |
とします。

## 自動周回

クリアしたダンジョンは生産施設のようにキャラクターを配置し、ダンジョン報酬を回収できます。どの素材を集めるかも生産施設と同じように人数を分配します。ダンジョンは生産施設のように強化要素がない代わりに、ダンジョンレベルによって報酬量が変動します。1ダンジョンレベルごとに+5%の報酬量になります。
自動周回した場合、獲得経験値はありません。
自動周回で収集できる素材は、配置したダンジョンレベルのDL帯に従います（現状は全5種がDL1から入手可能なため、どのDLでも全5種が対象です）。

## ダンジョン素材

ダンジョン素材は全5種ともDL1から入手できます。手動攻略の宝箱・敵ドロップ、および自動周回ともに同じプールから入手します。

| DL帯    | 入手できる素材                                       |
| ------- | ---------------------------------------------------- |
| DL 1〜  | 全5種(魔石・魔物の牙・魔物の皮・魔力結晶・古代歯車) |

自動周回では生産施設と同様に、プール内の素材に対してキャラクター数を配分します。未強化・1キャラクターあたりの基本生産量は以下のとおりです。

| 素材     | 価格 | 1分あたり |
| -------- | ---: | --------: |
| 魔石     |   5G |       0.5 |
| 魔物の牙 |  10G |      0.25 |
| 魔物の皮 |  10G |      0.25 |
| 魔力結晶 |  25G |       0.1 |
| 古代歯車 |  25G |       0.1 |

## ダンジョンレベル

ダンジョン名はレベルをそのまま表示します(例: ダンジョン Lv.1)。
攻略した次のレベルが解放されます。ただし挑戦できるダンジョンレベルはGR×10までに制限されます(GR1なら最大DL10)。10ダンジョンレベルごとにGRが1上昇します。ダンジョンレベル10をクリアするとGRは2になります。GRの上昇には魔力結晶と古代歯車がGR×50個ずつ必要です。GR1から2には各50個、9から10は各450個必要です。これらの素材はDL1から入手できるため、DL10クリアまでにGR上昇分を集められます。
